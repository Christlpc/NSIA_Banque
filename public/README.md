# Dossier Public

## Logo NSIA

Pour ajouter le logo NSIA, placez votre fichier logo dans ce dossier avec le nom `logo-nsia.png` ou `logo-nsia.svg`.

### Formats recommandés :
- **PNG** : avec fond transparent, taille recommandée 512x512px
- **SVG** : pour une meilleure qualité à toutes les résolutions

### Emplacement du logo :
Le logo sera automatiquement chargé dans :
- `components/layouts/AuthLayout.tsx` (page de login)
- Peut être utilisé dans d'autres composants si nécessaire

### Instructions :
1. Placez votre fichier logo dans ce dossier (`public/logo-nsia.png`)
2. Décommentez les lignes dans `components/layouts/AuthLayout.tsx` (lignes 23-30)
3. Commentez ou supprimez le placeholder (ligne 21)
